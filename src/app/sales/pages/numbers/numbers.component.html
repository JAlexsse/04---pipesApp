Decimal Pipes

Basic Angular pipes normally included in the Common Module.

Locale determines group sizing and separator, decimal point character, and other locale-specific configurations.

Format: num | number

Number without pipe: {{ num }}

Number with pipe: {{ num | number }}

You can pass an argument to determine what kind of format you desire on your number.

Format: num | number:'1.2-2'

Number without pipe: {{ num }}

Number with pipe: {{ num | number:'1.2-2' }}

Currency Pipes

Basic Angular pipes normally included in the Common Module.

Formatted according to locale rules that determine group sizing and separator, decimal-point character, and other locale-specific configurations.

Format: num | currency

Number without pipe: {{ num }}

Number with pipe: {{ num | currency }}

You can pass an argument to determine what kind of format you desire on your currency.

Format: num | currency:'EUR':'symbol-narrow':'1.4-4'

Number without pipe: {{ num }}

Number with pipe: {{ num | currency:'EUR':'symbol-narrow':'1.4-4' }}
